# 学习版高性能 Shiki 代码高亮库 - 产品说明

## 文档说明

### 1. 开发定位
基于 `TextMate 官方规范` 实现的**学习版高性能代码高亮 JS 库**，对标原生 Shiki 核心能力，核心学习目标为吃透「有限状态机 + 纯文本语法作用域解析」的完整核心逻辑，无 AST 语义解析，纯文本特征匹配实现高亮。

### 2. 核心宗旨
兼顾「极致学习价值」与「生产级可用」，核心解析逻辑无黑盒、可溯源，扩展能力灵活，解析性能对标专业高亮库。

### 3. 文档用途
作为完整开发落地手册，包含技术栈、文件结构、实现步骤、核心代码、扩展指南，所有内容均可直接落地开发。

## 产品特性

### 核心优势

#### 🎯 学习价值
- **透明化核心逻辑**：基于 TextMate 规范的有限状态机实现，无黑盒代码
- **可溯源设计**：从底层解析到上层渲染，完整可追踪

#### 🔧 易扩展
- **开闭原则**：对扩展开放，对修改关闭

#### 🌐 全平台兼容
- **浏览器环境**：Chrome/Firefox/Safari/Edge

## 技术亮点

### 1. 双层架构设计
- **底层核心解析层**：Golang + WASM 编译，处理 CPU 密集型解析逻辑
- **上层封装扩展层**：TypeScript，提供统一 API 和扩展能力

### 2. TextMate 规范实现
- **有限状态机**：支持基础状态和嵌套子状态（如 JSDoc 嵌套在多行注释中）
- **纯文本匹配**：无 AST 语义解析，保证与 VSCode/Shiki 逻辑一致
- **标准 Token 结构**：与 VSCode/Shiki 完全对齐

### 3. 灵活的扩展能力
- **多语言支持**：注册新语言无需修改核心代码
- **多主题支持**：主题切换无需重新解析代码
- **装饰器系统**：支持全局装饰和精准节点装饰

## 应用场景

### 适合使用场景

✅ **学习 TextMate 状态机原理**
- 想要深入理解代码高亮的底层实现
- 学习有限状态机在实际项目中的应用
- 掌握纯文本语法解析的核心逻辑

✅ **构建自定义代码高亮方案**
- 需要高性能的代码高亮库
- 需要灵活的扩展能力（自定义语言、主题、装饰器）
- 需要全平台兼容的解决方案

✅ **教学和演示**
- 作为前端库开发的实战项目
- 展示 Go + WASM + TypeScript 的技术栈应用
- 演示有限状态机的实际应用

### 不适合使用场景

❌ **需要完整语法分析**
- 需要代码语义分析（如类型检查、错误提示）
- 需要代码补全、重构等 IDE 功能
- 需要理解代码的抽象语法树（AST）

❌ **需要极致性能优化**
- 需要处理百万行级别的超大文件
- 需要毫秒级的实时响应
- 需要极致的内存优化

## 快速开始

### 安装依赖
```bash
bun install --dev typescript @types/node
bun install --dev jest @types/jest
```

### 编译项目
```bash
bun run build:ts
bun run build:wasm
```

### 基础使用
```typescript
import { codeToHtml } from './dist/ts/index.js';

const code = 'const x = 42;';
const html = await codeToHtml(code, 'javascript', 'dark');
console.log(html);
```
