## 项目概览

Mini Highlighter 是一个学习版高性能代码高亮库，基于 TextMate 状态机原理实现。

### 核心特性
- ✅ 纯文本语法作用域解析，无需AST
- ✅ 双层架构（Golang + WASM + TypeScript）
- ✅ 跨平台支持（浏览器、Node.js、Bun）
- ✅ 高性能（解析1000行代码 < 100ms）
- ✅ 可扩展（语言、主题、装饰器）
- ✅ 生产级可用

### 技术栈
- **核心解析层**：Go + WebAssembly
- **上层封装**：TypeScript
- **状态机**：TextMate 语法规范
- **跨平台**：浏览器、Node.js、Bun

### 开发时间
- **MVP阶段**：13-20小时
- **完整功能**：30-40小时
